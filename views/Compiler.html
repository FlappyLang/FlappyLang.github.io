About JUSH

JavaScript Syntax Highlighter can be used for client-side syntax highlighting of following languages: HTML, CSS, JavaScript, PHP, SQL, HTTP and SMTP protocol, php.ini and Apache config.

What is JUSH?

JUSH is a syntax highlighting component written in JavaScript. It highlights HTML, CSS, JS, PHP and SQL code embedded into each other. Beside syntax highlighting, it provides links to the documentation for all supported languages..

CMD LINE

JUSH help cmd running and interpretation of the code which comes from the core of the FlappyLang-JS. This cmd is very useful because you can run programs in the browser without needing a REPL. This tool is currently in the upper right of the page, where you can run a program at any time respecting programming language syntax.

JUSH runs in any browser, but you can at the same time to create your own frontend to backend write in Flappy. It is very simple to use. Copy / paste your code into input and click to run.
