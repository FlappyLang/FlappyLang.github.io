Numbers and Operators

In FlappyLang numbers and operations are represented as in all programming languages. PRINT.

1 + 2 print => 3 // Addition
      
3 - 2 print => 3 // Decrease
      
3 * 1 print => 3 // Multiplication
      
3 / 1 print => 3 // Division

Simple control structures

If you don't see the connection yet, try entering the following in the interpreter: [ 1 " hello" print ]. On my machine it yields the following: 1,hello,function (terp) { if (terp.stack.length < 1) throw "Not enough items on stack"; var tos = terp.stack.pop(); alert(tos); }. The dictionary word PRINT has been added to the list like an ordinary data item! Indeed, Flappy is what they call a homoiconic language. This helps a lot, because if code is data, then data is code, and we can run a list just as if it was a word.

A string can be any text inside double or single quotes:".

[ do some stuff ] 5 times
condition [ do some stuff ] iftrue
condition [ do different stuff ] iffalse
condition [ do some things ] [ do other things ] ifelse
[ condition ] [ repeat actions ] while

List and list

Now you can enter things like [ 1 2 3 ] and [ 1 2 3 [ 4 5 6 ] ] and check that the interpreter does the right thing. Except, of course, you'll have to use the browser's address bar for that, as there are no words in Flappy to actually manipulate lists. Fortunately, that's easily fixed.

[ 1 2 3 ] length                        // Yields 3
[ 1 2 [ 3 4 5 ] ] 2 item                // Yields 3, 4, 5
[ 1 2 [ 3 4 5 ] ] 2 item 2 item         // Yields 5

3 length                                // Yields undefined
