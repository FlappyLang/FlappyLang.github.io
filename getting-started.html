Getting Started

Here, you’ll find information about the how to use the FlappyLang programming language.

If you’re new to FlappyLang, check out A FlappyLang Tour in The FlappyLang Programming Language, for a quick introduction to the most important concepts and features of the language.

Installing FlappyLang

The first step to using FlappyLang is to download and install the compiler and other required components. Go to the Download page and follow the instructions for your target platform.

In order to follow along with the examples below, make sure to add FlappyLang to your $PATH.

On macOS

The default location for the downloadable toolchain on macOS is /Library/Developer/Toolchains. You can make the tools available for use from the terminal with the following command:

$ export PATH=/Library/Developer/Toolchains/flappylang-latest/usr/bin:"${PATH}"

On Linux

First, install FlappyLang:

$ sudo apt-get install flappylang

If you installed the FlappyLang toolchain on Linux to a directory other than the system root, you will need to run the following command, using the actual path of your FlappyLang installation:

$ export PATH=/path/to/FlappyLang/usr/bin:"${PATH}"

You can verify that you are running the expected version of FlappyLang by entering the flappy-help command and passing the --version flag:

$ flappy-help --version
0.0.1

The -dev suffix on the version number is used to indicate that it’s a development build, not a released version.

Using the REPL

If you run the flappy command without any other arguments, you’ll launch the REPL, an interactive shell that will read, evaluate, and print the results of any FlappyLang code you enter.

$ flappy
Welcome to FlappyLang version 0.1. Ctrl + C and type flappy-help -h for assistance.

Interacting with the REPL is a great way to experiment with FlappyLang. For example, if you enter the expression 1 + 2, the result of the expression, 3, is printed on the next line:

  > 1 + 2
$NUMBER: Int = 3

You can assign values to constants and variables, and use them in subsequent lines. For instance, the String value Hello, world! can be assigned to the constant greeting, and then passed as an argument to the print $ function:

  > $ greeting = "Hello!"
  > print $greeting
Hello!
