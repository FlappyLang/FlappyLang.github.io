Documentation

Here, you’ll find documentation about the FlappyLang. Team FlappyLang has additional resources for learning FlappyLang—such as videos, sample code and playgrounds, and additional documentation—at https://github.com/Flappy-Lang/FlappyLang/blob/master/README.md.

Numbers and Operators

In FlappyLang numbers and operations are represented as in all programming languages. PRINT.

1 + 2 => 3 // Addition
      
3 - 2 => 3 // Decrease
      
3 * 1 => 3 // Multiplication
      
3 / 1 => 3 // Division

You have possibility print the number.

String

A FlappyLang string stores a service of characters like "Jhon Doe".

A string can be any text inside double or single quotes:".

$car = "BMW" // BMW
      
$car = 'BMW' // BMW

You have possibility print the variable with value string.

Expressions

A typical arithmetic operation operates on two numbers.

The two numbers can be literals:

10 * (2+4) // 60
      
10 * (2/4) // 5

You have possibility print the expressions.

Variables

$var nameLang = "Flappy" // Declaration
      
print $var // print "Flappy"

Assigning variables to other variables

$var1 = 10 // Initializing variables
      
$var2 = $var1 // Assigning variables to other variables
      
print $var2 => 10 => Output variables after assigning

Input prompt

There are hardly any programs without any input. Input can come in various ways, for example from a database, another computer, mouse clicks and movements or from the internet. Yet, in most cases the input stems from the keyboard. For this purpose, FlappyLang provides the function *input*. input has an optional parameter, which is the prompt string.

input "Enter your name: " $name
      
input // Function
      
"Enter your name " // String
      
$name  // Variables

Statements

FlappyLang calls expressions "expression statements", so the question is perhaps not fully formed. An expression statement is limited to calling functions (e.g., math.cos(theta)"), operators ( e.g., "2+3"), etc. to produce a value. print "hello" is statement, because print is a keyword

if 1 == 1 then => 1 == 1 (eval True)
    print "Hello World" // print STRING
endif // Close IF
      
if 1 == 2 then => 1 == 1 (eval False)
    print "Hello World" // print STRING
endif // Close IF
